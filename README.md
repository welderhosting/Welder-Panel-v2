# Welder-Panel

Made by walksys

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Welder-Panel
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the application:
   ```bash
   npm run build
   ```

4. Create an admin user:
   ```bash
   npm run createuser
   ```

5. Start the server:
   ```bash
   npm run start
   ```

## Development

To run the panel in development mode with auto-reloading:

```bash
npm run dev
```
